Steps to Install

Rito Client

Navigate to the Riot Client official website and download the installer. Run the installer and follow the on-screen instructions to complete the installation process. Once installed, launch the Riot Client application.

Vanguard

Vanguard is a security system used by Riot Games to protect the integrity of their games. Upon launching Riot Client for the first time, you will be prompted to install Vanguard. Accept the prompt and follow the installation.

Valorant

Select Valorant from the Riot Client library, click "Install," and wait for the download and installation process to complete. Once finished, you can launch Valorant and start playing.
